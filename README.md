# 🎵 Ponto Musical

🔗 [Acesse o projeto online aqui](https://juliademartini.github.io/pontomusical/)

O **Ponto Musical** é um projeto de interface web desenvolvido para partituras online compartilhadas por qualquer pessoa.

---

## 🚀 Tecnologias Utilizadas

Para o desenvolvimento desta interface, utilizei:

* **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
* **CSS3:** Estilização personalizada, layouts com Flexbox e Grid.
* **Design Responsivo:** Interface adaptada para dispositivos móveis, tablets e desktop.

---

## 🛠️ Funcionalidades

* **Vitrine de Instrumentos:** Exibição organizada de produtos/serviços.
* **Design Intuitivo:** Navegação focada na experiência do usuário (UX).
* **Layout Adaptável:** Visualização otimizada em qualquer tamanho de tela.
* **Identidade Visual:** Focada no nicho musical, com elementos modernos.
